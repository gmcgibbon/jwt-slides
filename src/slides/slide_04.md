## Is there a Gem for it?

Yes, several.

```ruby
gem 'jwt'               # https://github.com/jwt/ruby-jwt
gem 'knock'             # https://github.com/nsarno/knock
gem 'json_web_token'    # https://github.com/garyf/json_web_token
gem 'json-jwt'          # https://github.com/nov/json-jwt
gem 'devise_token_auth' # Token-based, but not actually JWT
                        # https://github.com/lynndylanhurley/devise_token_auth
```
